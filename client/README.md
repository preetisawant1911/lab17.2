This is a simple MERN (MongoDB, Express, React, Node.js) based Task Manager project.

## How to run the project

### Backend (Server)
1. Go to server folder:
   cd server
2. Install packages:
   npm install
3. Create .env file:
   MONGO_URI=your_mongodb_url
   PORT=3001
4. Start server:
   npm start

### Frontend (Client)
1. Go to client folder:
   cd client
2. Install packages:
   npm install
3. Create .env file:
   REACT_APP_API_URL=http://localhost:3001
4. Start React app:
   npm start

## Features
- Add tasks
- View tasks
- Fully working MERN connection
